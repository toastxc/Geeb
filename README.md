# Geeb
C# based Jellyfin client for music
## Features
- [x] Login to Jellyfin
- [x] Fetch volumes
- [x] Fetch albums
- [x] Download songs
- [x] Song playback (flac, mp3)
- [x] media buttons (play, pause, skip, stop)
- [ ] Display playback info
- [ ] Display current song info
- [ ] Artist Display
